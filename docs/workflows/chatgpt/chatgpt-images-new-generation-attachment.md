# ChatGPT Images: New Generation With Attachment

- JSON: `workflows/chatgpt/chatgpt_images_new_generation_attachment_v1.json`
- Purpose: Open `chatgpt.com/images`, upload one local file or image, send an image-generation prompt, default to 2 variations unless already specified, and return the resolved `chat_id`.
- Required params: `message_text`, `attachment_file_path`
- Optional params: `variation_count`
