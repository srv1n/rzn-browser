- JSON: `workflows/chatgpt/chatgpt_send_current_composer_v1.json`
- Purpose: Reuse the current ChatGPT tab without navigating, optionally verify an already attached file/image, send the prompt from the visible composer, and return the resolved `chat_id`.
- Required params: `message_text`
- Optional params: `attachment_file_path`
