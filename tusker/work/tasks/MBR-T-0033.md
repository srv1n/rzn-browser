# MBR-T-0033 · MB-034 add skip-safe browser native-host smoke tests

## Intent

Add skip-safe Playwright smoke tests that can verify Chrome, Edge, Chromium, and simultaneous Chrome + Edge native-host bridge connectivity when local browser binaries and built runtime binaries are available.

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | Chrome smoke verifies extension/native-host connection and `browser targets` Chrome record when enabled. | E2E smoke spec |
| A2 | Edge smoke verifies extension/native-host connection and `browser targets` Edge record when Edge is available. | E2E smoke spec |
| A3 | Chromium smoke verifies extension/native-host connection and `browser targets` Chromium record when Chromium is available. | E2E smoke spec |
| A4 | Simultaneous Chrome + Edge smoke verifies two bridge records are visible. | E2E smoke spec |
| A5 | Simultaneous smoke opens/routes one Chrome session and one Edge session without cross-routing. | E2E smoke spec |
| A6 | Tests print doctor output on failure. | E2E smoke spec/source check |
| A7 | CI/local runs skip unavailable browser binaries with clear reasons. | E2E smoke spec/skipped run |

## Non-goals

- Forcing Edge or Chrome installation in CI.
- Running native-host smoke tests by default on every unit-test pass.
