# ADI-T-0002 · Meta Ad Library creative asset capture (video/image download)

## Intent

Given ads enumerated by ADI-T-0001, resolve and download the actual creative assets — the videos and images an agency analyst would study. Videos are the high-value target: the downstream analysis pipeline (rzn-tools ADS-T-0001) transcribes and vision-describes them to extract hooks and offers.

Resolution strategy: prefer CDN URLs already present in the DOM; fall back to network-response interception when video variants only appear at play time. Downloads write to `assets/<ad_id>-<n>.<ext>` next to the manifest, and the manifest's `assets[].path` entries are filled in.

## Dependencies

- ADI-T-0001 — this task consumes its manifest and extends the same capture run.

## Scope

In:
- Asset URL resolution for image, video, and carousel ads (carousel → multiple assets per ad).
- Download with size cap and per-asset timeout; failures recorded per-ad in the manifest (`error` note) without aborting the run.
- Idempotent re-runs: existing valid files are skipped; content hash recorded per asset for dedupe across ads reusing one creative.
- Manifest update in place: `assets[].path`, byte size, content hash.

Out:
- Any transcoding or thumbnail generation (analysis side owns processing).
- Watermark removal or any transformation of creatives.
- Non-Meta sources (Google Transparency Center capture lands with ADI-T-0003 if its previews allow).

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | For a live capture of a video-heavy advertiser, ≥90% of video/image ads end up with local assets. | Live proof run capsule with counts. |
| A2 | Manifest `assets[].path` entries point at existing files; hashes and sizes recorded. | Validation script over the run output. |
| A3 | Re-running the capture skips already-downloaded assets. | Second run capsule showing skip counts. |
| A4 | Size caps and timeouts enforced; a failing asset yields a per-ad error record, not a run abort. | Focused test with an unreachable/oversized fixture URL. |

## Non-goals

- No guarantee for DRM-protected or expiring CDN URLs beyond one retry.
- No bulk archiving of entire ad libraries — caps from ADI-T-0001 bound the run.
