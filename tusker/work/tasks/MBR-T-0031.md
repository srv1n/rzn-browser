# MBR-T-0031 · MB-032 add native host installer regression tests

## Intent

Add focused native-host installer regression tests covering browser-specific manifest paths, origin normalization, invalid input rejection, and uninstall isolation without touching the real user home or registry.

| ID | Outcome | Proof |
|---|---|---|
