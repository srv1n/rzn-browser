# MBR-T-0036 · Productize make install and extension bundle build

## Intent

Make the top-level install flow real for Chrome, Edge, and Chromium.

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | `make install` resolves to a release install path that builds Rust binaries, builds extension bundles, installs binaries, registers the native host, and prints load-unpacked directories. | Focused command or dry-run inspection plus source review |
| A2 | Extension build output is canonical at `extension/dist/{chrome,edge,chromium}` with manifests, runtime JS, popup assets, icons, and `rzn-build.json`. | Extension build/test |
| A3 | Extension dependency install is lockfile-based. | Lockfile presence and install command inspection |

## Non-goals

- Published store packaging.
- Firefox native messaging.

## Verification

| Covers | Check | Result | Notes |
|---|---|---|---|
| A1 | `make -n install` | pass | Install target resolves to `bash ./install.sh`; shell syntax checks for `setup.sh`, `install.sh`, and `extension/build.sh` passed |
| A1 | `cargo check -p rzn-browser -p rzn-native-host` | pass | Check completed with existing unused-code warnings |
| A2 | `PATH="$HOME/.bun/bin:$PATH" bun --bun run build` from `extension/` | pass | Generated `extension/dist/{chrome,edge,chromium}` and legacy Chrome alias |
| A2 | `find extension/dist ...` plus required-file checks | pass | Manifest, `rzn-build.json`, background worker, and icons exist for Chrome/Edge/Chromium |
| A3 | `extension/bun.lock` and `bun install --frozen-lockfile` | pass | Locked install checked 136 packages with no changes |
| A2 | `bun x playwright test tests/e2e/content.spec.ts --reporter=line` | pass | Focused extension e2e loaded `extension/dist/chrome` and verified content script injection |
