# MBR-T-0016 · MB-017 bind browser sessions to browser instances and active bridges

## Intent

Bind browser sessions to resolved browser instances and active bridges so session-scoped commands keep using the same browser while multiple browsers are connected.

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | Opening a session with `browser: edge` stores a session record with session ID, browser instance ID, bridge ID, creation time, and last activity time. | Focused supervisor test |
| A2 | Later session-scoped `execute_step` routes to the same browser instance without repeating the browser target. | Focused supervisor test |
| A3 | An explicit target that conflicts with the session binding returns `SESSION_TARGET_CONFLICT`. | Focused supervisor test |
| A4 | Closing a session removes the session binding. | Focused supervisor test |
| A5 | Multiple sessions can bind to different browser instances and run side by side without cross-routing. | Focused supervisor test |

## Non-goals

- Composite tab references. That starts in MB-020.
- Full reconnect policy for same browser instance. That is MB-019.
