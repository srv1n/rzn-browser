# MBR-T-0041 · Repair macOS installed CLI signing

## Intent

Make macOS bundle/runtime installs repair locally signed `rzn-browser` and `rzn-native-host` binaries during atomic install so AppleSystemPolicy does not SIGKILL the CLI before `main` runs.

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | Installed macOS CLI is no longer killed by code-signature validation before printing help. | `rzn-browser --help` passes from PATH after reinstall/hotfix. |
| A2 | Packaged macOS installers repair invalid or ad-hoc signatures during binary install without replacing future non-ad-hoc signatures unnecessarily. | `bash -n scripts/bundle/install-macos.sh` and `sh -n scripts/release/install-runtime.sh` pass. |
| A3 | Browser workflow smoke succeeds through both Chrome and Edge bridge targets. | `rzn-browser run google search --browser chrome --param search_query="rust"` and `rzn-browser run google search --browser edge --param search_query="rust"` pass. |

## Non-goals

- Not changing workflow JSON semantics.
- Not changing browser extension/native-host protocol behavior.

## Verification

| Covers | Check | Result | Notes |
|---|---|---|---|
| A1 | rzn-browser --help | pass | PATH command executes after replacing the poisoned installed binary with a freshly built, re-signed release binary. |
| A2 | bash -n scripts/bundle/install-macos.sh | pass | Bundle installer parses after adding local/ad-hoc signature repair. |
| A2 | sh -n scripts/release/install-runtime.sh | pass | Release installer parses after adding local/ad-hoc signature repair. |
| A3 | rzn-browser run google search --browser chrome --param search_query="rust" | pass | Chrome run returned Google result payload through supervisor/native-host bridge. |
| A3 | rzn-browser run google search --browser edge --param search_query="rust" | pass | Edge run returned Google result payload through supervisor/native-host bridge. |
