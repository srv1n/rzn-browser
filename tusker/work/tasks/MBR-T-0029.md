# MBR-T-0029 · MB-030 fix extension per-browser build outputs

## Intent

Make the extension build pipeline produce repeatable per-browser outputs for Chrome, Edge, and Chromium under `extension/dist/<target>` with target-specific compiled build metadata and CI-friendly package scripts.

## Acceptance

| ID | Outcome | Proof |
|---|---|---|
| A1 | `bun run build:chrome`, `build:edge`, and `build:chromium` produce valid `dist/<target>` outputs. | Command verification |

| A3 | Each output compiles with target-specific `RZN_EXTENSION_TARGET` metadata. | Command/source verification |
