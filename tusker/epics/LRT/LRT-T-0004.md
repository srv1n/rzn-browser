# LRT-T-0004 · Fold browser worker MCP surface into rzn-browser mcp browser

## Intent
Preserve MCP browser compatibility while retiring `rzn-browser-worker` as a separate durable daemon.

## Read this when

Read this for task scope, acceptance, evidence, or implementation history.


## Scope

In:
- Extract current browser worker MCP tool list/dispatch into reusable code.
- Add `rzn-browser mcp browser` stdio mode.
- Route MCP tool calls to supervisor IPC.
- Keep tool names/result shapes stable where possible.

Out:
- Replacing MCP with app-specific APIs.
- Serving HTTP MCP unless needed after stdio works.
