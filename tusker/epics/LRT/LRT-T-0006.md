# LRT-T-0006 · Migrate rzn-browser CLI execution onto supervisor client

## Intent
Move CLI workflow execution from worker endpoint attach/spawn to the supervisor client while preserving current command UX.

## Read this when

Read this for task scope, acceptance, evidence, or implementation history.


## Scope

In:
- Make `rzn-browser run ...` call supervisor by default.
- Auto-start or heal supervisor where safe.
- Keep legacy worker attach/spawn as compatibility fallback during migration.
- Preserve output shape used by scripts.

Out:
- Removing existing workflow catalog commands.
- Changing workflow JSON contracts.
